# ②下载 Sakurairo 主题

你可以从以下平台下载到 Sakurairo 主题

  • GitHub : https://github.com/mirai-mamori/Sakurairo (opens new window)
  • Gitee : https://gitee.com/mirai-mamori/Sakurairo (opens new window)

# ③安装 Sakurairo 主题

# 方式一

进入仪表盘，点击左侧菜单的外观选项

点击主题标题旁的添加

点击添加主题标题旁的上传主题

然后在下图的上传主题中，选中 Sakurairo 主题

选择文件-现在安装

# 方式二

直接将你下载好的压缩包上传放在网站根目录 /wp-content/themes/下解压

解压完成后进入 WordPress 后台- > 外观 -> 主题- > 启用 Sakurairo 主题 即可
